Self-Driving Car Part 2: UNET

I developed an AI model that helps a self-driving car understand its surroundings by identifying different objects in an image. This is done using a technique called image segmentation, where the AI learns to highlight important areas, like cars, roads, and pedestrians. I used a special type of neural network called UNET, which is great at recognizing objects in images. This project was built using Python and PyTorch, which are tools that help train AI models to see and understand images like a human would.

Technologies: Python, PyTorch, NLP

AI generated Art

I created an AI-powered tool that takes a regular photo and transforms it into a painting in the style of a famous artist. This works by using a technique called "neural style transfer," where a computer learns the patterns and colors of an artwork and applies them to another image. Imagine taking a photo of a sunset and making it look like it was painted by Van Gogh! I built this using Python and PyTorch, which help the AI understand and recreate artistic styles.

Technologies: Python, PyTorch, NLP
